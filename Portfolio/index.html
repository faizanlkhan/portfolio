Research

Learning Adaptive Feature-Weight Optimization and Landing-Site Prediction for Lunar Surface Exploration Using Reinforcement Learning

  • Built a PPO-based RL model to learn adaptive feature weights on a 5058×5058 lunar grid, replacing fixed MCDM weighting.
  • Improved suitability-prediction stability and consistency by 18–22% over baseline methods.
  • Generated 60m suitability and AI-weight maps and extracted top landing zones via 95th-percentile filtering.
  • Achieved stable convergence (~10⁴ steps) with mean suitability/reward 0.74 and max suitability 0.9066.
  • Implemented complete preprocessing: normalization, raster harmonization, slope/hazard inversion, and geospatial outputs.

Real-Time Driver Drowsiness Detection with Temporal Alert System

  • Developed a real-time drowsiness detection system using YOLOv5 & OpenCV for webcam-based inference.
  • Collected and annotated a custom dataset (~200+ images) using LabelImg for supervised object detection (awake vs drowsy).
  • Trained a YOLOv5 model (80 epochs, batch size 8), achieving mAP@0.5 ≈ 0.99 and mAP@0.5:0.95 ≈ 0.80+.
  • Implemented temporal filtering using consecutive frame analysis to reduce false positives and improve prediction stability.
  • Designed a real-time alert mechanism that triggers continuous audio feedback when drowsiness persists for ≥3 seconds.

Fake News Detection using NLP with Feature-Based Interpretability

  • Developed an NLP-based classification system using TF-IDF and Logistic Regression on ~45,000 news articles, covering preprocessing, feature extraction, and evaluation.
  • Achieved ~98.7% accuracy (validated on held-out test data) with balanced precision and recall.
  • Evaluated model performance against a Multinomial Naive Bayes baseline, with Logistic Regression achieving higher accuracy due to better feature weighting.
  • Analyzed model coefficients to identify high-impact words influencing predictions and understand class-specific patterns.
  • Examined model limitations, identifying reliance on stylistic patterns and reduced performance on short or context-limited inputs.
